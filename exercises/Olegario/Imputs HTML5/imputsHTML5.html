Nuevos Imputs En HTML5

Datalist

Facilita el autocompletado de un campo tipo input.

<input list="listaInternos">
<datalist id="listaInternos">
   <option value="Olegario">
   <option value="Miguel">
   <option value="Isaac">
   <option value="Eddy">
   <option value="Beto">
   <option value="Mayra">
   <option value="Axel">
   <option value="Sara">
   <option value="Gerardo">
</datalist>

Búsqueda (search)

Al ingresar texto en el campo, el navegador muestra una "x" a la derecha para borrar todo lo que hemos escrito.

<input type=“search” name=“busqueda”>

Teléfono (tel)

A la hora de completar este desde un smartphone, convierte tu teclado a números de teléfono.

<input type=“tel” name=“telefono”>

Dirección url (url)

En este campo, el teclado del smarphone es qwerty pero en modo “url”, ofrece teclas fundamentales para escribir una dirección web como el punto, la barra “/” o la tecla “.com”.

<input type=“url” name=“url”>

Correo electrónico (email)

Con este imput, el teclado del smartphone es qwerty pero también muestra la tecla “@”.

<input type=“email” name=“correo”>

Fecha y hora (datetime)

Al cliquear en el campo verás un calendario muy completo que el navegador dispone de forma nativa.

<input type=“datetime” name=“fechahora”>

Fecha (date)

Muestra el calendario nativo del navegador

<input type="date" name="fecha">

Mes (month)

El calendario aquí permite seleccionar el número de mes.

<input type="month" name="mes">

Semana (week)

Permite elegir el número de semana del año.

<input type="week" name="semana">

Hora (time)

Muestra el formato de hora, con los dos puntos “:” correspondientes y las flechas para subir o bajar el horario.

<input type="time" name="hora">

Número (number)

Para este input tenemos los atributos min y max para establecer el máximo y el mínimo que acepta el campo.
En un smartphone vemos el teclado numérico

<input type="number" name="num" min="0" max="50">

Rango (range)

Presenta como un control para arrastrar con el mouse (o con el dedo en un móvil con pantalla táctil). Este campo también acepta los atributos min y max.

<input type="range" name="rango" min="0" max="50">

Color (color)

Se presenta de forma nativa un selector de color… otra funcionalidad que comunmente tendríamos que hacer con javascript.

<input type="color" name="color">
