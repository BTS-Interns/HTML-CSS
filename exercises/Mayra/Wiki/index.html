Definition of HTML

HTML is a programming language used for developing websites. This is the acronym for HyperText Markup Language, ie HyperText Markup Language, which could be translated as Language for Hypertext Document Format.

It is an open format that emerged from the SGML (Standard Generalized Markup Language) tags is understood as a system that allows you to sort and label various documents within a list. This language is used to specify the names of the labels to be used when ordering, there are no rules for the organization.

Definition of CSS

CSS is a language used in the presentation of HTML documents. An HTML document is being usually "a website". It serves to organize the presentation and appearance of a web page. This language is mainly used by web browsers and computer internet web programmers to choose many display options such as colors, font types and sizes, etc.

Did you know the source HTML dates back to 1980, when physicist Tim Berners-Lee, working at CERN (European Organization for Nuclear Research) proposed a new system of "hypertext" to share documents this means that had as use a language for physicists to share their research.
