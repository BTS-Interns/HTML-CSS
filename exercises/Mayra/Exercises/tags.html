Metadatos del documento

Los metadatos contienen información sobre la página. Esto incluye información sobre estilos,scripts y datos que ayudan al software (search engines, browsers, etc.) a usar y generar la página. Los metadatos de estilos y scripts pueden estar definidos en la página o estar enlazados a otro fichero que contiene la información.

Elementos Interactivos

HTML ofrece una selección de elementos que pueden ayudar a crear objetos de interfaz de usuario interactivos.

Element Description

<details>

es usado como un widget de revelación a través del cual el usuario puede obtener información adicional .

representa una caja de diálogo u otro componente interactivo, como inspector o ventana. Los elementos <form> pueden integrarse dentro de un diálogo especificándolos con el atributo method="dialog". Componentes Web

Los Componentes Web son una tecnología relacionada con HTML que hacen que sea posible, en esencia, crear y personalizar elementos como si fueran HTML normal. Además, pueden crear versiones personalizadas de los elementos HTML estándar.

Nota: Los elementos de los Componentes Web están definidos en World Wide Web Consortium (W3C) Web Components collection of specifications en lugar de la especificación HTML. Además, la especificación de los Componentes Web no ha sido finalizado y está sujeto a cambios.

Element Description

<content>

es usado dentro de un Shadow DOM como un insertion point . No está pensado para ser usado en HTML ordinario . Es usado con Web Components.

<element>

es usado dentro de un Shadow DOM como un insertion point . No está pensado para ser usado en HTML ordinario . Es usado con Web Components.

<shadow>

elemento se utiliza como un punto de inserción DOM sombra. Es posible utilizarlo si se han creado múltiples raíces de sombra bajo una sombra de acogida. No es útil en HTML ordinario. Se utiliza con componentes web.

<template>

es un mecanismo para la celebración de contenido del lado del cliente que no se va a dibujar cuando se carga una página, pero posteriormente puede ser instanciada durante el tiempo de ejecución usando JavaScript.
